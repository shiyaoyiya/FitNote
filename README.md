# FitNote 健身记录小程序

FitNote 是一个专为健身爱好者设计的小程序，帮助用户记录和追踪健身训练情况。通过简洁直观的界面，用户可以轻松管理训练动作、创建训练模板、记录每日训练数据，并查看历史记录。

## 功能特点

### 1. 日历视图
- 直观的月度日历界面，方便查看和记录每日训练情况
- 支持左右滑动切换月份
- 点击日期可查看或编辑当天的训练记录
- 长按日期可清空当天记录

### 2. 动作管理
- 添加和管理自定义训练动作
- 查看动作历史记录和训练频率
- 长按动作可删除不需要的动作

### 3. 模板系统
- 创建训练模板，包含多个动作组合
- 为模板设置颜色标识，便于区分不同类型的训练
- 快速应用模板到指定日期，提高记录效率

### 4. 数据记录
- 记录每次训练的详细数据（重量、组数、次数等）
- 支持有氧运动和休息日记录
- 自动计算训练总量，便于追踪训练强度变化

### 5. 数据统计
- 周训练量统计和对比
- 年度训练数据总览
- 训练备忘录功能，记录训练计划和心得

### 6. 纪念日功能
- 添加和管理健身相关的重要纪念日
- 自动计算距离纪念日的天数

### 7. 主题模式
- 支持深色和浅色模式切换
- 根据系统设置自动适配主题

## 技术架构

- 基于 UniApp 框架开发，支持多端运行
- 使用 Vue3 语法
- 采用 Pinia 进行状态管理
- 数据存储使用本地存储方案

## 主要页面

1. **首页日历** - 主要的训练记录界面
2. **日训练详情** - 记录和查看某天的详细训练数据
3. **动作历史** - 查看特定动作的历史记录
4. **模板详情** - 管理训练模板的动作和设置
5. **年度总览** - 查看年度训练数据统计
6. **训练备忘录** - 记录训练计划和心得

## 安装和使用

1. 克隆项目代码
2. 安装依赖：`npm install`
3. 在 HBuilderX 中导入项目
4. 编译运行到目标平台（微信小程序、H5等）

## 项目结构

```
FitNote/
├── pages/                 # 页面文件
│   ├── index/             # 首页相关页面
│   ├── actionHistory/     # 动作历史页面
│   ├── templateDetail/    # 模板详情页面
│   ├── year/              # 年度总览页面
│   └── help/              # 帮助/备忘录页面

│   └── settings.js        # 设置相关状态
├── static/                # 静态资源
└── ...
```
